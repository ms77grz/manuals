===PACMAN===

Установка пакета:
# pacman -S emacs

Обновить все мои пакеты предварительно синхронизировав их базу данных:
# pacman -Syu

Синхронизация базы данных пакетов:
# pacman -Sy

Принудительная синхронизация, в том случае, если после -Sy прошло относительно короткое время:
# pacman -Syy

Обновление пакетов:
# pacman -Su

Настоятельно рекомендуется запускать сочетание -Syu для обновления всех пакетов:
Кроме того для принудительной синхронизации и обновления необходимо запускать -Syyu

Подготовка обновлений для их последующей выборочной установки:
# pacman -Syyuw

Например:
# sudo apt-get update && apt-get upgrade == pacman -Sy && pacman -Su == pacman -Syu





Поиск пакета в официальных репозиториях
S yay -Ss mplayer

Просмотр информации о пакете
$ yay -Si mplayer

Обновление всех пакетов из AUR и официальнвх репозиториев
$ yay -Syu
